# sps30-utils

Debian / Raspbian utils for the Sensirion SPS30 air quality sensor.

# Warnings

In an early alpha state!

# Requires

The open-source Sensirion example code.  A debian operating system.  A UART wired up to the sps30

# Modules required 

## Raspberry pi

In /etc/modules add:

```
crc8
industrialio
sps30
```
